# PRD — NEXSKILLS (Nova Versão)

## 1. Visão do Produto

O NEXSkills é uma plataforma centralizada de "skills" (comandos, MCPs, automações e ferramentas) para uso com Claude Code, permitindo que usuários descubram, instalem e utilizem rapidamente funcionalidades prontas.

Objetivo: transformar o projeto atual (HTML estático) em uma aplicação moderna, escalável e interativa.

---

## 2. Problema

Hoje:

* Interface estática (HTML + CSS)
* Difícil manutenção
* Sem autenticação
* Sem escalabilidade
* Componentes não reutilizáveis

Usuário precisa:

* Encontrar skills rapidamente
* Copiar comandos facilmente
* Organizar por categorias
* Ter acesso autenticado (área de membros)

---

## 3. Objetivos

### Objetivo principal

Criar uma dashboard moderna, performática e escalável.

### Objetivos secundários

* Melhorar UX/UI
* Implementar login
* Criar estrutura modular
* Facilitar expansão futura

---

## 4. Público-alvo
